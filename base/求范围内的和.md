# 求范围内的和

- 题目描述：
  - 我们会传递给你一个包含两个数字的数组。返回这两个数字和它们之间所有数字的和。
  - 最小的数字并非总在最前面。

- 解题思路：
  - 先把最大数最小数区分开来，再用for循环将中间的数字补全，最后reduce计算和。

``` js
function sumAll(arr) {
  //最小值和最大值
  var min = Math.min.apply(null,arr);
  var max = Math.max.apply(null,arr);
  
  //推入数组中间的数
  for(var i = min+1; i < max; i++){
    arr.push(i);
  }
  
  //返回总和
  return arr.reduce(function(prev,next){
    return prev + next;
  });

}

sumAll([1, 4]);
```
